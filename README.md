# roblox-starter

塔防守公主 · Rojo + Luau，最多 4 人好友房。

## 快速开始

```powershell
cd 项目根目录
mise trust
mise install
mise run serve          # 保持运行
```

Studio：Baseplate → Rojo **Connect** → **Sync In** → **Play**。
